About this site

This site began as a single jQuery-powered button hitting the Sunlight Congress API, and has grown to include the Google Maps Javascript API and lots of vanilla JS. The map's data layer was built using data from the U.S. Census Bureau, converted to GeoJSON with Mapshaper, and populated with representative data using the Congress API once again.

If you have feature suggestions or bug reports, please raise an issue on GitHub.

About Sister District Project

SDP is a grassroots initiative to pair solidly blue congressional districts with flippable red ones, with the goal of bringing sanity to the House in 2018. They're supporting lots of critical state and local races this year, so check them out!
